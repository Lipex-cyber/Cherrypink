🍒 Cherry Pink By Mari

E-commerce desenvolvido para a loja Cherry Pink By Mari, especializada em produtos de maquiagem, beleza e acessórios femininos.

🛍 Sobre o Projeto

A Cherry Pink é uma loja online criada com foco em:

Visual moderno e feminino

Identidade visual em tons rosa e branco

Layout responsivo (Mobile e Desktop)

Experiência de navegação simples e intuitiva

🎯 Objetivo

Desenvolver uma loja virtual elegante, leve e funcional para:

Expor produtos de maquiagem

Divulgar promoções

Direcionar clientes para redes sociais

Fortalecer presença digital da marca

🚀 Funcionalidades

Página inicial com banner promocional

Filtro de produtos por categoria

Layout adaptado para dispositivos móveis

Estrutura organizada de produtos

Integração com Instagram

Design otimizado para conversão

🛠 Tecnologias Utilizadas

HTML5

CSS3

JavaScript

Estrutura responsiva com Flexbox e Grid

Hospedagem via GitHub Pages

📱 Responsividade

O site foi desenvolvido com foco em:

📲 Smartphones

💻 Desktop

📟 Tablets

🎨 Identidade Visual

Cores principais: Rosa, Branco e detalhes brilhantes

Elementos visuais delicados e femininos

Estilo moderno e atrativo
